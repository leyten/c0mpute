# Referrals

Invite someone to c0mpute and earn **5% of the USDC value of every prompt they pay for**. Forever, for as long as they use it.

## How it works

1. Grab your link from [settings → Referrals](https://c0mpute.ai/settings#referrals). It looks like `c0mpute.ai/r/yourcode`.
2. Share it. Anyone who lands on it can try c0mpute immediately — no account needed. The link stays valid in their browser for 30 days.
3. When they sign up, they're bound to you. Permanently — referrals never expire or get reassigned.
4. From then on, every job they **pay credits for** (chat, image generation, API usage) sends 5% of its USD value to your balance.

## What pays and what doesn't

| Usage | Pays you 5%? |
|---|---|
| Paid chat prompts (pro / max) | yes |
| Paid image generation | yes |
| Paid API usage | yes |
| Free daily prompts | no |
| Staker-allowance (stake-to-use) credits | no |

Only **self-paid** usage counts. If it didn't cost your referral anything, it doesn't pay you anything.

## Where the 5% comes from

Worker pay is untouched. The 5% comes out of the protocol's own cut, so the revenue split on a referred job is:

- **70 / 25 / 5** — worker / treasury / you (standard)
- **80 / 15 / 5** — worker / treasury / you (when the worker has a staked boost)

## Getting paid

Referral earnings land in the same pending balance as worker earnings. Withdraw any time from [settings](https://c0mpute.ai/settings#worker) — minimum $1.00, sent as USDC on Solana.

## The fine print

- One referrer per account, set at signup, permanent.
- Self-referral doesn't work — your own code can't bind to you.
- Existing accounts can't be claimed; only new signups bind.
