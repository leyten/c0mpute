# Getting started

## 1. Login

Go to [c0mpute.ai](https://c0mpute.ai) and click **Login with Privy**. You can sign in with:

- Email (magic link)
- Crypto wallet (Phantom, MetaMask, etc.)
- Social accounts (Google, Discord, Twitter)

No lengthy sign-up. Just authenticate and you're in.

## 2. Pick a tier

Before sending a message, select your model tier from the dropdown:

- **Free** — Qwen 1.5B. Fast, basic, runs in browser workers. Good for simple questions.
- **Pro** — Dolphin Mistral 7B. Higher quality, uncensored, runs in browser workers.
- **Max** — Qwen2.5 14B. Best quality, uncensored, web search enabled. Runs on native workers.

See [Model tiers](/user-guide/model-tiers) for a detailed comparison.

## 3. Send a message

Type your message in the chat box and hit send. Your request enters the job queue and gets matched to an available worker.

- If a worker is available, you'll see tokens streaming back within seconds
- If no workers are available for your tier, your request queues until one comes online
- Responses stream in real-time — you see each token as it's generated

## 4. That's it

No API keys. No credit card. No usage limits on Free tier. Just type and get answers from the decentralized network.

For higher quality and uncensored responses, use Pro or Max tier. For web-grounded answers with citations, use Max.
